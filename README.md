# mushroom-dataset
A JSON dataset filled with mushroom data. 

Follow the structure as seen if you want to comitt data.

Create a merge request and it'll be added as long as you follow the structure and the data is correct.

Structure for each mushroom can be seen as is right now.
The following values are required:
<ul>
<li>name_lat</li>
<li>name_eng</li>
<li>color</li>
<li>images</li>
<li>All the ecoligcal fields</li>
<li>All the fields inside classification</li>
</ul>

**All fields are strings unless specified as something different**<br>
**Colors are hexadecimal strings**
